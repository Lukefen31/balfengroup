# Nathan Fenech Concreting - nathanfenech.com

Malta's concreting specialist. Australian standards, Maltese ground.

## Project Structure

```
nathanfenech/
  index.html        # Main landing page (multi-page SPA)
  images/
    nfc-logo.svg    # Brand logo
    pour-hero.jpg   # Hero background image
    pour-1.jpg      # First pour Malta - June 2026
    pour-2.jpg
    pour-3.jpg
    pour-4.jpg
```

## Stack

Plain HTML/CSS/JS — single file, no build step, no dependencies except Google Fonts.

## To deploy

Upload `index.html` + `images/` folder to any static host (Cloudflare Pages, Netlify, etc.)
or drop it into the public_html folder on a cPanel host.
